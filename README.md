ShadowHost removes the complexity of managing server `.bat` files and command lines, replacing them with a sleek, high-performance dashboard. Control your world with a single click.

---

## ‚ö° Key Features

* **One-Click Start:** Launch your server instantly without touching a terminal.
* **Smart Detection:** Automatically detects your installed Java version and system RAM to ensure compatibility.
* **Live Console:** Watch your server logs in real-time with a clean, readable log viewer.
* **Modern UI:** A beautiful, dark-themed interface designed for clarity and ease of use.
* **Native Performance:** Ultra-lightweight and fast. It doesn't hog resources that your server needs.
